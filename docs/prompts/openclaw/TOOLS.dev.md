# TOOLS.md - 用户工具笔记（可编辑）

这个文件用于记录你关于外部工具和约定的**个人笔记**。
它不定义有哪些工具；OpenClaw 会在内部提供内置工具。

## 示例

### imsg

- 发送 iMessage/SMS：说明收件人/内容，发送前确认。
- 消息要简短；避免发送秘密。

### sag

- 文本转语音：说明声音、目标扬声器/房间，以及是否流式播放。

把你希望助手了解的本地工具链细节都写在这里。
