# TOOLS.md - 本地笔记

技能定义**工具如何工作**。这个文件记录**你环境的具体细节** —— 只属于你的那部分。

## 适合放这里的内容

比如：

- 摄像头名称与位置
- SSH 主机与别名
- TTS 偏好声音
- 扬声器/房间名称
- 设备昵称
- 任何环境相关的细节

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，触发式监控

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## 为什么要单独存放

技能是共享的；你的环境是独特的。分开存放可以让你更新技能不丢笔记，也能在共享技能时不泄露基础设施。
